## About HackerCamp

HackerCamp is WCCOMPS's intensive summer cybersecurity training program designed to provide students with comprehensive, hands-on security education. Unlike our competitions, HackerCamp focuses on skill development, mentorship, and career preparation in a collaborative learning environment.

## Program Structure

### Duration & Schedule

- **When:** July (4 weeks intensive program)
- **Format:** In-person with some remote options
- **Schedule:** Monday-Friday, 9:00 AM - 5:00 PM
- **Location:** Partner university campus (varies by year)

### Daily Schedule

**Morning Sessions (9:00 AM - 12:00 PM)**
- Technical lectures and demonstrations
- Tool training and walkthroughs
- Skill-building exercises
- Guest speakers from industry

**Afternoon Sessions (1:00 PM - 5:00 PM)**
- Hands-on labs and projects
- Team challenges and exercises
- Mentorship sessions
- Project work time

**Optional Evening Activities**
- CTF competitions
- Social events and networking
- Career development workshops
- Guest lectures and panels

## Curriculum

### Week 1: Foundations

**Network Security Basics**
- TCP/IP fundamentals
- Network protocols and services
- Packet analysis with Wireshark
- Network scanning and reconnaissance

**System Administration**
- Linux command line mastery
- Windows Server management
- Service configuration and hardening
- Scripting and automation basics

### Week 2: Offensive Security

**Penetration Testing**
- Vulnerability assessment
- Exploitation techniques
- Privilege escalation
- Post-exploitation activities

**Web Application Security**
- OWASP Top 10 vulnerabilities
- SQL injection and XSS
- Authentication and session attacks
- Secure coding practices

### Week 3: Defensive Security

**Incident Response & Forensics**
- Incident detection and triage
- Digital forensics fundamentals
- Log analysis and correlation
- Malware analysis basics

**Security Operations**
- SIEM configuration and use
- Threat hunting techniques
- Detection engineering
- Security automation

### Week 4: Specialization & Projects

**Choose Your Path:**
- Advanced penetration testing
- Cloud security (AWS/Azure)
- DevSecOps and CI/CD security
- Industrial control systems security
- Mobile security

**Capstone Project:**
- Team-based security project
- Real-world scenario
- Presentation to industry judges
- Portfolio piece

## Eligibility

### Who Should Apply?

HackerCamp is ideal for:
- College students (undergrad and grad)
- Career-changers entering cybersecurity
- IT professionals expanding to security
- Competition team members seeking training
- Self-taught enthusiasts building credentials

### Prerequisites

- **Basic Technical Skills** - Comfortable with computers and networking
- **Programming Experience** - At least one scripting language helpful
- **Time Commitment** - Available for full 4-week program
- **Enthusiasm** - Passionate about learning cybersecurity

### Application Process

1. **Submit Application** - Forms open in March
2. **Essay Question** - Why cybersecurity interests you
3. **Technical Assessment** - Basic skills evaluation
4. **Interview** - Brief conversation with coordinators
5. **Selection** - Notifications sent in April
